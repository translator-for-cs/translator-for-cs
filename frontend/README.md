# Frontend

## Project setup
Install node, latest version is best version.

Install yarn, then do.
```
cd frontend
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
